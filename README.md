# Web Application for Residential Facility Administration

A full-stack Hostel Management System developed to streamline and automate hostel operations such as student management, room allocation, complaint handling, and outing request management. The system provides separate dashboards for students and administrators, improving transparency and operational efficiency.



##  Project Overview

Traditional hostel management processes are often manual and time-consuming. This project provides a centralized digital platform that enables students and hostel administrators to manage hostel-related activities efficiently.

## The application supports:
- Student Information Management
- Room Allocation Management
- Complaint Registration and Tracking
- Outing Request Management
- Mess Information Management
- Role-Based Authentication (Admin & Student)


## Features

### Student Module
- Student Login(By Using Regd No. & Password)
- View Personal Details
- View Room Information
- View Room Allocation Details
- Submit Complaints
- Track Complaint Status
- Apply for Outing Requests
- View Outing History
- View Mess Menu
- View Doctor Availability
- Change Password


###  Admin Module
- Admin Login(By Using Email & Password)
- Register Student
- Search Student Records
- Manage Room Allocation
- View and Resolve Complaints
- Approve/Reject Outing Requests
- Monitor Hostel Activities
- View Outing Reports and Mess Records
- Manage Worker's Details
- Manage Doctor Availability
- View Student Medical Records and Prescriptions
- Change Password
